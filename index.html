01

CPU / ซีพียู

CPU ย่อจากมาก Central Processing Unit หรือชื่อเรียกในภาษาไทยคือหน่วยประมวลผลกลาง

เป็นส่วนประกอบหลักของคอมพิวเตอร์ที่ทำหน้าที่ประมวลผลและควบคุมมีหน้าที่รับผิดชอบในการดำเนินคำสั่งและจัดการข้อมูลในระบบคอมพิวเตอร์

02

RAM / แรม

RAM ย่อจากมาก Random-access memory หรือชื่อเรียกในภาษาไทยคือหน่วยความจำเข้าถึงโดยสุ่ม

แรมคือหน่วยความจำชั่วคราวที่คอมพิวเตอร์ใช้จัดเก็บข้อมูลที่จำเป็นเพื่อจะได้ดึงนำข้อมูลนั้นมาใช้ได้อย่างรวดเร็ว

เนื้อหาแหล่งที่มาของ CPU เนื้อหาแหล่งที่มาของ RAM
