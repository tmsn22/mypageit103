รายการอุปกรณ์คอมพิวเตอร์

ลำดับการประกอบเครื่องจากอุปกรณ์

  1. เตรียมเคส + อุปกรณ์ + จัดโต๊ะให้โล่ง
  2. ติดตั้ง CPU ลงบนเมนบอร์ด เปิดก้านล็อก วาง CPU ให้ตรงมุม ห้ามออกแรงกดและล็อกกลับ
  3. ทาซิลิโคนลงบน CPU แต้มตรงกลางเม็ดเดียวพอ
  4. ติดตั้งชุดระบายความร้อน ยึดน็อตให้ทั่ว เสียบสายพัดลมให้ตรงสาย CPU_FAN
